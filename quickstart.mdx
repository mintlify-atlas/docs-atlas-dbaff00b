## Installation

Install Codemogger globally via npm:

```bash
npm install -g codemogger
```

Or run it directly with `npx` without installing:

```bash
npx codemogger --help
```

## Index Your First Codebase

<Steps>
  <Step title="Navigate to your project">
    Open your terminal and navigate to any codebase you want to search:
    ```bash
    cd ~/projects/my-app
    ```
  </Step>

  <Step title="Index the codebase">
    Run the index command to parse and embed your code:
    ```bash
    codemogger index .
    ```
    
    This creates a `.codemogger/index.db` file in your project. Indexing is incremental - subsequent runs only reprocess changed files.
  </Step>

  <Step title="Verify indexing">
    List all indexed files to confirm:
    ```bash
    codemogger list
    ```
  </Step>
</Steps>

## Search Your Code

Codemogger supports two search modes:

<CodeGroup>
```bash Semantic Search
# Natural language queries - find concepts across files
codemogger search "authentication middleware"
codemogger search "HTTP request parsing and response writing"
codemogger search "how are errors handled?"
```

```bash Keyword Search
# Precise identifier lookup - find exact function/class/type names
codemogger search "BTreeCursor" --mode keyword
codemogger search "handleRequest" --mode keyword
```
</CodeGroup>

### Search Options

Customize search results with these flags:

| Flag | Default | Description |
|------|---------|-------------|
| `--mode` | `semantic` | Search mode: `semantic`, `keyword`, or `hybrid` |
| `--limit` | `5` | Maximum number of results to return |
| `--format` | `json` | Output format: `json` or `text` |
| `--snippet` | `false` | Include full code snippet in output |

Example with options:

```bash
codemogger search "database connection pool" --limit 10 --snippet --format text
```

## MCP Integration

Connect Codemogger to your AI coding agent (Claude Code, OpenCode, etc.) via MCP:

<Steps>
  <Step title="Locate your MCP config file">
    Find your agent's MCP configuration file:
    
    <CodeGroup>
    ```bash Claude Code
    ~/Library/Application Support/Claude/claude_desktop_config.json
    ```
    
    ```bash OpenCode
    ~/.config/opencode/mcp.json
    ```
    </CodeGroup>
  </Step>

  <Step title="Add Codemogger server">
    Add the Codemogger MCP server to your config:
    
    ```json
    {
      "mcpServers": {
        "codemogger": {
          "command": "npx",
          "args": ["-y", "codemogger", "mcp"]
        }
      }
    }
    ```
  </Step>

  <Step title="Restart your agent">
    Restart your coding agent to load the MCP server. You'll now have access to three tools:
    
    - **`codemogger_search`** - Search indexed codebases (semantic & keyword)
    - **`codemogger_index`** - Index a new codebase
    - **`codemogger_reindex`** - Update index after file changes
  </Step>
</Steps>

<Note>
  The MCP server uses the same `.codemogger/index.db` file created by the CLI. Any codebase you index with the CLI is immediately searchable via MCP.
</Note>
